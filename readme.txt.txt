In raw_files is :

- the css stylesheets for these pages

In networking is all that is needed to run the site and its features on the localhost:3000.
- install git bash (https://git-scm.com/download/win)
- open the git bash terminal
- type the root path of where you stored the GreenLightCaskIndex package followed by the in package directions...
$ nodemon "Drive:\dirnames__*\GreenLightCaskIndex\networking\app.js"*
(The site's now up.)
- open your browser
- type in the search localhost:3000
- you can now try sending yourself an email in the subscription box down below.
